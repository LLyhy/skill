# OpenClaw Skills Collection

A collection of useful skills for OpenClaw.

## Skills List

### 安全必备
- skill-vetter - 安全审计官
- session-log - 操作监控

### 效率工具
- tavily-search - 实时联网搜索
- self-improving-agent - 自我进化

### 办公必备
- file-manager-pro - 本地文件管家
- notion-sync - Notion双向同步

### 创作开发
- doc-summarizer - 文档总结器
- code-reviewer - 代码审查工具
- xhs-content-generator - 小红书文案生成器

## Installation

Import using OpenClaw's remote import feature with this repository.
